What is FRIDAY ADE?

FRIDAY ADE is a VS Code-style IDE powered by OpenClaw — an autonomous AI development engine inspired by FRIDAY and JARVIS from Iron Man. OpenClaw doesn't just assist — it acts. It creates files, modifies code, manages your GitHub, and can even rewrite its own source code.

Create, read, write, and delete any file or folder.

Installation

🪟 Windows
🍎 macOS
🐧 Linux
  1. Install Node.js

    Download and run the Windows installer from nodejs.org. Choose the LTS version. Make sure "Add to PATH" is checked during install.

  2. Install Ollama

    Download from ollama.ai and run the installer. Ollama will start automatically in the system tray.

  3. Extract FRIDAY ADE

    Right-click FRIDAY-ADE-v1.0.zip → Extract All → choose a permanent location like C:\Apps\friday-ade

  4. Run Setup

    Open the extracted folder and double-click:

    scripts\setup.bat

    This installs all dependencies, pulls AI models, and starts FRIDAY ADE.

  5. Create Desktop Shortcut

    Double-click the shortcut creator to add FRIDAY ADE to your Desktop and Start Menu:

    scripts\create-shortcut-windows.bat
  6. Pull AI Models (optional, recommended)

    Open Command Prompt or PowerShell and run:

    ollama pull llama3.2
ollama pull deepseek-r1
ollama pull codellama

Build a Real Installer

You can package FRIDAY ADE into a proper installer (.exe, .dmg, .AppImage) using electron-builder.

# Install electron-builder first
npm install

# Windows → .exe installer
npm run build:win

# macOS → .dmg disk image
npm run build:mac

# Linux → .AppImage + .deb
npm run build:linux

# All platforms at once
npm run build:all

Built installers appear in the dist/ folder.

Troubleshooting

⚠ npm install fails
Make sure Node.js 18+ is installed: node --version. If below 18, reinstall from nodejs.org.
⚠ Electron not found
Run npm install from inside the friday-ade folder, not the scripts folder.
⚠ Ollama models not responding
Make sure Ollama is running: ollama serve in a terminal. OpenClaw talks to Ollama on port 11434.
✓ QUICK START (no Ollama)
FRIDAY ADE works without Ollama — it falls back to cloud AI automatically. You can start building immediately and add Ollama models later.
